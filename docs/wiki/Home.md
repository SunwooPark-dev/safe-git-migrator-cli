# Safe Git Migrator CLI Wiki

Welcome to the canonical wiki for `safe-git-migrator-cli`.

This CLI is a Windows-first npm tool for safely ingesting MIT-licensed GitHub or Hugging Face repositories and migrating reusable agent-oriented assets into target environments such as:
- OMX
- Codex App
- Hermes
- Antigravity
