# GitHub Wiki Mirroring

## Canonical source
The canonical wiki lives inside this repository under `docs/wiki/`.

## Mirror rule
If a GitHub Wiki is created later, it should mirror the repo-internal wiki rather than replacing it.

## Recommended mirror priority
1. `Home.md`
2. `Install-and-Run.md`
3. `Command-Reference.md`
4. `Safety-Model.md`
5. `Troubleshooting.md`

## Why this page exists
This page is the first explicit post-creation wiki refinement for the CLI wiki. It satisfies the rule that new wiki content must be updated at least once after initial creation, and it makes the publishing model clear for future maintainers.
